# Mythiq UI - Creative AI Platform Frontend

## 🌟 Overview

Mythiq UI is the beautiful, professional frontend for the Mythiq AI creative platform. It provides a unified interface for AI chat, game generation, and media creation - all with a stunning dark theme and glassmorphism design.

## ✨ Features

### 🎨 **Beautiful Design**
- Modern dark theme with purple gradients
- Glassmorphism effects with backdrop blur
- Smooth animations and hover states
- Mobile-responsive design

### 🧭 **Navigation**
- **Home Dashboard** - Stats and service overview
- **AI Chat** - Intelligent conversation interface
- **Game Creator** - Game generation tools
- **Media Studio** - Image, video, and audio creation

### 📊 **Dashboard Features**
- Real-time stats tracking
- Service status indicators
- Quick start examples
- Professional branding

## 🚀 Quick Start

### **Development**
```bash
npm install
npm run dev
```

### **Production Build**
```bash
npm run build
npm run preview
```

### **Deploy to Railway**
1. Push to GitHub repository
2. Connect to Railway
3. Auto-deploys with zero configuration

## 🛠️ Technology Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Beautiful icons
- **shadcn/ui** - Professional UI components
